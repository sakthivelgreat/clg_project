Two-Way ANOVA

Two-Way ANOVA tests if there are significant differences between the means of groups split on two independent variables and if there is an interaction effect between the variables.

The formulas for the sums of squares are:

\[ \text{SST} = \sum_{i=1}^{a} \sum_{j=1}^{b} \sum_{k=1}^{n} (X_{ijk} - \bar{X})^2 \]

\[ \text{SSA} = \sum_{i=1}^{a} n b (\bar{X}_{i\cdot\cdot} - \bar{X})^2 \]

\[ \text{SSB} = \sum_{j=1}^{b} n a (\bar{X}_{\cdot j \cdot} - \bar{X})^2 \]

\[ \text{SSAB} = \sum_{i=1}^{a} \sum_{j=1}^{b} n (\bar{X}_{ij \cdot} - \bar{X}_{i \cdot \cdot} - \bar{X}_{\cdot j \cdot} + \bar{X})^2 \]

\[ \text{SSE} = \sum_{i=1}^{a} \sum_{j=1}^{b} \sum_{k=1}^{n} (X_{ijk} - \bar{X}_{ij \cdot})^2 \]

The degrees of freedom are:

\[ \text{df}_A = a - 1 \]

\[ \text{df}_B = b - 1 \]

\[ \text{df}_{AB} = (a - 1)(b - 1) \]

\[ \text{df}_E = ab(n - 1) \]

The mean squares and F-statistics are:

\[ \text{MSA} = \frac{\text{SSA}}{\text{df}_A} \] \[ F_A = \frac{\text{MSA}}{\text{MSE}} \]

\[ \text{MSB} = \frac{\text{SSB}}{\text{df}_B} \] \[ F_B = \frac{\text{MSB}}{\text{MSE}} \]

\[ \text{MSAB} = \frac{\text{SSAB}}{\text{df}_{AB}} \] \[ F_{AB} = \frac{\text{MSAB}}{\text{MSE}} \]

\[ \text{MSE} = \frac{\text{SSE}}{\text{df}_E} \]
